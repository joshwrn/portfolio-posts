## Overview

The React Dial component is a customizable dial that can be used to select a value within a range. The dial can be rotated by dragging the mouse or by using touch gestures on mobile devices. The dial can be customized with various props to control its appearance and behavior.

I'm going to walk through the process of creating the React Dial component and explain how it works.

### Getting the quadrant

We will need to determine the degrees of rotation based on the position of the mouse or touch event. We can do this by calculating the angle between the center of the dial and the current position of the mouse or touch event. 

It's not as simple as just calculating the angle between two points. We need to take into account the quadrant that the mouse or touch event is in. This is because the angle between two points is not unique. For example, the angle between the points <Math math="(1, 0)"/> and <Math>(0, 1)</Math> is 90 degrees, but so is the angle between the points <Math>(1, 0)</Math> and <Math>(0, -1)</Math>. 

<ExcaliSvg path="lab/dial/Untitled-2022-11-05-0138_o1fk9v.svg" width={470} />

We also don't want the dial to instantly jump to 360 if we're starting at 0. So we need just check to see which quadrant we're inbefore we calculate the angle.

<CodeSnippet
language="tsx"
title="The function to convert a point in a quadrant to degrees"
code={`const getDegFromQuadrant = (
  point: { x: number; y: number },
  deg: number,
  lastQuadrant: number,
  setLastQuadrant: React.Dispatch<React.SetStateAction<number>>
): number => {
  const q1 = point.x >= 0 && point.y >= 0;
  const q2 = point.x <= 0 && point.y >= 0;
  const q3 = point.x <= 0 && point.y <= 0;
  const q4 = point.x >= 0 && point.y <= 0;
  if (q1 || q4) {
    if (lastQuadrant === 2) {
      return -90 - deg;
    }
    setLastQuadrant(q1 ? 1 : 4);
    return 270 - deg;
  }
  if (q2 || q3) {
    if (lastQuadrant === 1) {
      return 450 + deg;
    }
    setLastQuadrant(q2 ? 2 : 3);
    return 90 + deg;
  }
  return 0;
};`}
/>

### Closest Point on the Dial

Using basic trigonometry, we can get the closet point on the unit circle using the center of the dial and the current mouse position.

After we get the mouse position and the center of the dial, we subtract the mouse position from the center point. `{ x: centerPoint.x - mouse.x, y: centerPoint.y - mouse.y }`.

Next, we calculate the sine and cosine. `cos = point.x / Math.sqrt(Math.pow(point.x, 2) + Math.pow(point.y, 2))`. To get the sine, we do the same thing but with the y value.

This is the formula to get the point.

<Math block math="\left( \frac{x}{\sqrt{x^{2} + y^2}}, \frac{y}{\sqrt{x^2 + y^2}} \right)" />

To get the degrees we calculate the angle using the `asin` function and convert it to degrees. `theta = Math.asin(sin)` and `degrees = toDeg(theta)`.

<Math block math="\theta = \sin^{-1} \left( \frac{y}{\sqrt{x^2 + y^2}} \right)" />

Lastly, we need to check which quadrant we're in and convert the point to degrees. We do this by using the `getDegFromQuadrant` function we created earlier.

<CodeSnippet
language="tsx"
title="The function to get the closest point on the dial"
code={`const turnDial = (f: DraggableEvent, d: DraggableData) => {
  const e = f as MouseEvent
  if (!dialRef.current) return
  const ref = dialRef.current
  const box = ref.getBoundingClientRect()
  const centerPoint = {
    y: box.top + box.height / 2,
    x: box.left + box.width / 2,
  }
  const mouse = { x: e.clientX, y: e.clientY }
  const point = { x: centerPoint.x - mouse.x, y: centerPoint.y - mouse.y }
  const sin = point.y / Math.sqrt(Math.pow(point.x, 2) + Math.pow(point.y, 2))
  const cos = point.x / Math.sqrt(Math.pow(point.x, 2) + Math.pow(point.y, 2))
  const theta = Math.asin(sin)
  const degrees = getDegFromQuadrant(
    { x: -cos, y: -sin },
    toDeg(theta),
    lastQuadrant,
    setLastQuadrant
  )
  const increase = increment * Math.floor(degrees / increment)
  setDegrees(increase > max ? max : increase < min ? min : increase)
}`}
/>

import React, { useRef, useState } from "react";

import type { DraggableData, DraggableEvent } from "react-draggable";
import { DraggableCore } from "react-draggable";

const toDeg = (rad: number) => {
  return rad * (180 / Math.PI);
};

const getDegFromQuadrant = (
  point: { x: number; y: number },
  deg: number,
  lastQuadrant: number,
  setLastQuadrant: React.Dispatch<React.SetStateAction<number>>
): number => {
  const q1 = point.x >= 0 && point.y >= 0;
  const q2 = point.x <= 0 && point.y >= 0;
  const q3 = point.x <= 0 && point.y <= 0;
  const q4 = point.x >= 0 && point.y <= 0;
  if (q1 || q4) {
    if (lastQuadrant === 2) {
      return -90 - deg;
    }
    setLastQuadrant(q1 ? 1 : 4);
    return 270 - deg;
  }
  if (q2 || q3) {
    if (lastQuadrant === 1) {
      return 450 + deg;
    }
    setLastQuadrant(q2 ? 2 : 3);
    return 90 + deg;
  }
  return 0;
};

export const Dial: FC<{
  max?: number;
  min?: number;
  increment?: number;
  initial?: number;
  children?: JSX.Element;
  degrees: number;
  setDegrees: React.Dispatch<React.SetStateAction<number>>;
}> = ({
  max = 360,
  min = 0,
  increment = 45,
  degrees = 0,
  setDegrees,
  children,
}) => {
  const dragRef = useRef(null);
  const dialRef = useRef<HTMLDivElement>(null);
  const [lastQuadrant, setLastQuadrant] = useState(2);

  const turnDial = (f: DraggableEvent, d: DraggableData) => {
    const e = f as MouseEvent;
    if (!dialRef.current) return;
    const ref = dialRef.current;
    const box = ref.getBoundingClientRect();
    const centerPoint = {
      y: box.top + box.height / 2,
      x: box.left + box.width / 2,
    };
    const mouse = { x: e.clientX, y: e.clientY };
    const point = { x: centerPoint.x - mouse.x, y: centerPoint.y - mouse.y };
    const sin =
      point.y / Math.sqrt(Math.pow(point.x, 2) + Math.pow(point.y, 2));
    const cos =
      point.x / Math.sqrt(Math.pow(point.x, 2) + Math.pow(point.y, 2));
    const theta = Math.asin(sin);
    const degrees = getDegFromQuadrant(
      { x: -cos, y: -sin },
      toDeg(theta),
      lastQuadrant,
      setLastQuadrant
    );
    const increase = increment * Math.floor(degrees / increment);
    setDegrees(increase > max ? max : increase < min ? min : increase);
  };

  return (
    <div className="wrapper">
      <div className="outer">
        <div className="dial-container">
          <div
            className="handle-container"
            style={{
              transform: \`rotate(\${degrees}deg)\`,
            }}
          >
            <div className="handle" />
          </div>
          <div className="child-wrapper">{children}</div>
          <DraggableCore onDrag={turnDial} ref={dragRef}>
            <div className="dial" ref={dialRef} />
          </DraggableCore>
        </div>
      </div>
    </div>
  );
};
`,
"App.tsx": `import { useState } from "react";
import { styled } from "styled-components";
